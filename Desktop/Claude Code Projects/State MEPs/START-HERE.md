# Session Startup Checklist

When opening Claude Code for this project:

1. Say: "Read PROJECT.md, CURRENT.md, and GITHUB-WORKFLOW.md. Summarize where we are."
2. Say: "Create a new branch for [today's work]" (if starting new work)
3. Review the summary and decide what to work on

When ending session:
1. Say: "Update CURRENT.md with status and Git Status section"
2. Say: "Push everything to GitHub"
